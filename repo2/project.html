Projects by Team SJR

City-Sathi

City-Sathi is a smart city companion web application designed to help users explore nearby hostels, restaurants, hospitals, and markets within their city. The project focuses on a clean UI and smooth user experience with light and dark modes.

It includes a fixed navigation bar, login/sign-up popups, and a floating Team SJR button for a dynamic team view.

Technologies Used: HTML5, CSS3, and JavaScript (LocalStorage for theme switching).

Future Enhancements: Weather & Map APIs, dynamic search, and database-based authentication.
